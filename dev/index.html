The CSS Overflow Property

What is the CSS Overflow Property?

In CSS, very element on a page is rendered in a rectangular box. But what if the content meant to go inside the box can't fit? The CSS overflow property allows you to specify how you would like that handled.

The Overflow property has 4 values:

  • - visible (the default property)
  • - hidden
  • - scroll
  • - auto

visible

with this property, the content goes beyond the box.

(this is usually not the move)

CODE
  • .box{
  • overflow: visible;
  • }

hidden

This property literally hides any content that extends beyond the box.

WARNING: the content hidden this way is completely inaccessible

CODE

  • overflow: hidden;
  • }

Scroll

This hides the content outside the box, but it offers scrollbars to scroll inside the box and view the content.

Keep in mind that you will always get both horizontal + vertical scrollbars no matter what, even if the content requires only one scrollbar.

CODE
  • .box{
  • overflow: scroll;
  • }
